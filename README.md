# AI YouTube Video Summarizer

## Overview
This project is a simple AI-powered tool that summarizes YouTube videos into structured insights.

The system helps users quickly understand long videos by generating:

- Short Summary
- Detailed Summary
- Key Bullet Points
- Actionable Insights

## How It Works
1. User provides a YouTube link or transcript.
2. If a link is provided, the system guides the user to generate a transcript using a free AI tool.
3. The transcript is processed using AI to generate structured summaries.

## Tools Used
- Botpress (chatbot interface)
- AI language models for summarization
- Prompt engineering
